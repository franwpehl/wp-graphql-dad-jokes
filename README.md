# WPGraphQL Dad Jokes

This is an extension to the WPGraphQL plugin (https://github.com/wp-graphql/wp-graphql) that adds a `dadJoke` root query
that returns a random dad joke from `icanhazdadjoke.com`

To query for a dad joke:

```
{
  dadJoke
}
```
